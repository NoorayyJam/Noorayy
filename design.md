# Portfolio Website - Design Style Guide

## Design Philosophy

### Visual Language
Clean, professional, and trustworthy design that immediately communicates reliability and expertise. The aesthetic prioritizes clarity over complexity, ensuring small business owners can quickly understand the services and feel confident in reaching out.

### Color Palette
- **Primary Background**: Pure white (#FFFFFF) and light gray (#F9FAFB)
- **Text Color**: Dark gray (#111827) for maximum readability
- **Accent Color**: Deep navy blue (#14213D) - professional and versatile
- **Supporting Colors**: Various shades of gray for subtle elements

### Typography
- **Primary Font**: Inter (sans-serif) - modern, clean, highly readable
- **Heading Hierarchy**: Large, bold headings with ample white space
- **Body Text**: Smaller, optimized for mobile reading
- **Consistent Spacing**: Proper line height and letter spacing throughout

## Visual Effects & Styling

### Used Libraries
- **Anime.js**: Subtle scroll animations and button interactions
- **Splide.js**: Smooth portfolio image carousel
- **ECharts.js**: Clean data visualization (if needed)
- **Tailwind CSS**: Utility-first styling framework

### Animation Effects
- **Scroll Animations**: Gentle fade-in effects for sections
- **Button Hover**: Subtle color transitions and slight scale effects
- **Card Interactions**: Minimal hover states that don't distract
- **Loading States**: Smooth transitions between content

### Header Effect
- **Fixed Navigation**: Clean, minimal header with smooth scroll
- **Active States**: Subtle highlighting of current section
- **Mobile Menu**: Slide-out navigation with backdrop blur

### Styling Approach
- **Mobile-First**: All elements designed for mobile then enhanced for desktop
- **Consistent Spacing**: 8px grid system for perfect alignment
- **Professional Shadows**: Subtle depth without being distracting
- **High Contrast**: 4.5:1 minimum ratio for accessibility

### Image Treatment
- **Hero Image**: Professional workspace photo showcasing web design
- **Portfolio Images**: Clean mockups of completed projects
- **Service Icons**: Simple, professional iconography
- **Consistent Aspect Ratios**: Unified visual presentation

### Button Design
- **Primary CTA**: Deep navy blue background with white text
- **Secondary Actions**: Outlined style with navy blue border
- **Hover States**: Smooth color transitions
- **Touch-Friendly**: Minimum 44px touch targets

### Layout Principles
- **Single Column**: Mobile-optimized vertical layout
- **Generous White Space**: Breathing room between elements
- **Clear Hierarchy**: Visual importance through size and spacing
- **Consistent Alignment**: Left-aligned text, centered elements where appropriatepriate
